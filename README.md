# Neway RP Bot — обновление прав доступа

Discord-бот модерации и системы лицензий. Все команды начинаются с префикса `-`.

Ниже — краткое описание того, что изменилось в этом обновлении, и полный
список команд с правами на их использование.

---

## Что изменилось

### 1. `-del` и `-delus` — новая система доступа по ролям и каналам

Раньше эти команды были доступны фиксированному списку ролей в любом канале.
Теперь доступ зависит от роли **и** от того, в каком канале вызвана команда.

| Группа ролей | Где можно использовать |
|---|---|
| «Супер-модераторы» — `1520321736673853512`, `1520321736673853511`, `1520321736673853509`, `1520321736648949932`, `1520321736673853508` | **Везде** на сервере, без ограничений |
| Роли с исключением — `1520321736673853510`, `1520321736673853505` | Только в каналах категории `1520456833599279205`, **кроме** каналов категории `1520456097545191425` |
| Роли без исключения — `1520321736648949936`, `1520321736648949935` | Только в каналах категории `1520456833599279205`, **включая** каналы категории `1520456097545191425` |
| Все остальные | Доступа нет |

Всё настраивается в `config.py` (`DELETE_GLOBAL_ROLE_IDS`, `DELETE_EXEMPT_ROLE_IDS`,
`DELETE_NO_EXEMPT_ROLE_IDS`, `DELETE_CATEGORY_ID`, `DELETE_EXEMPT_CATEGORY_ID`).

### 2. `-license`, `-cancel`, `-data` — теперь только в одном канале

Эти три команды теперь можно вызывать **только** в канале
`1521214107469221999`. Исключение — участники с правом **Администратор**
на сервере, им можно из любого канала.

Настраивается константой `LICENSE_COMMANDS_CHANNEL_ID` в `config.py`.

### 3. `-help` обновлён

Справка по `-del`, `-delus`, `-license`, `-cancel`, `-data` теперь показывает
актуальные ограничения по ролям и каналам.

### 4. 🛠 Новое: кнопочная админ-панель `-admin` (BETA)

Полноценная панель управления ботом на кнопках Discord — не нужно помнить
синтаксис текстовых команд. Всё в одном сообщении, переключение разделов —
через редактирование этого же сообщения.

> 🧪 **Функция находится в BETA-тестировании.** Возможны ошибки и изменения
> в будущем. Каждый экран панели явно помечен как BETA.

**Доступ к команде `-admin`:** только роли `1520321736673853512`,
`1520321736673853511`, `1520321736673853509`, `1520321736648949932`.

**Как это работает:**
- Панель открывается одним сообщением с кнопками разделов.
- **Управлять кнопками может только тот, кто вызвал `-admin`** — если
  нажмёт кто-то другой, бот молча (эфемерно, видно только ему) откажет.
- Ввод данных (ID/упоминание пользователя, ID канала, тип лицензии, текст
  заметки) — через всплывающее окошко Discord (Modal), а не через чат:
  ничего не остаётся в канале, само окно закрывается после отправки.
- Панель автоматически «истекает» через **10 минут** после вызова —
  кнопки становятся неактивными, нужно вызвать `-admin` заново.
- Внутри каждого раздела действуют **те же права**, что и у обычных
  текстовых команд (например, выдавать лицензии на панели может только
  тот, у кого есть роль для `-license`) — панель не даёт доступа больше,
  чем обычные команды, а просто предоставляет удобный интерфейс к ним.

**Разделы панели:**

| Раздел | Соответствует команде | Требуемые роли |
|---|---|---|
| 🗃 База данных | `-data` | роли `-data` |
| 📜 Выдать лицензию | `-license` | роли `-license` |
| 🚫 Отозвать лицензию | `-cancel` | роли `-cancel` |
| 📝 Заметка | `-note` | роли `-data` |
| 🧾 Wipe истории | `-wipe` | только Администратор |
| 🗑 Удалить сообщения | `-del` | см. таблицу `-del`/`-delus` выше |
| 🧹 Удалить сообщения юзера | `-delus` | см. таблицу `-del`/`-delus` выше |
| 📖 Справка | — | всем, кто открыл панель |

Логи всех действий, сделанных через панель, пишутся в те же каналы логов,
что и для обычных текстовых команд (с пометкой «через админ-панель»).

---

## Что НЕ менялось

Все существующие текстовые команды (`-report`, `-license`, `-cancel`,
`-data`, `-note`, `-wipe`, `-del`, `-delus`, `-help`, `-bot`) работают
как раньше — админ-панель является дополнительным интерфейсом к ним,
а не заменой.

---

## Полный список команд

### 🚩 `-report @пользователь`
Отправляет жалобу на участника в канал администрации с последними сообщениями
нарушителя и данными профиля.
**Доступ:** всем участникам сервера.
**Кулдаун:** 60 сек (7 сек — для ролей `1520321736673853512`, `1520321736673853511`, `1520321736673853510`).

### 📜 `-license @пользователь <1|2> <ID_канала>`
Выдаёт лицензию РП (тип 1 — Участник, тип 2 — Менеджер) и права на
указанный канал-форум внутри категории `1520456833599279205`.
**Доступ:** роли `1520321736673853512`, `1520321736673853511`, `1520321736673853509`.
**Канал вызова:** только `1521214107469221999` (админам — можно из любого).

### 🚫 `-cancel @пользователь <1|2>`
Отзывает все активные лицензии указанного типа: снимает роль, убирает права
с каналов, помечает лицензии в БД как отозванные.
**Доступ:** роли `1520321736673853512`, `1520321736673853511`, `1520321736673853510`,
`1520321736673853504`, `1520321736673853503`, `1520321736648949936`,
`1520321736648949935`, `1520321736648949932`.
**Канал вызова:** только `1521214107469221999` (админам — можно из любого).

### 🗃 `-data [запрос]`
Поиск по базе лицензий: по ID лицензии, упоминанию, Discord ID или никнейму.
**Доступ:** роли `1520321736673853512`, `1520321736673853511`, `1520321736673853510`,
`1520321736673853505`, `1520321736648949936`, `1520321736648949935`.
**Канал вызова:** только `1521214107469221999` (админам — можно из любого).

### 📝 `-note @пользователь <текст>`
Добавляет/заменяет заметку персонала к пользователю в базе лицензий.
**Доступ:** те же роли, что и у `-data`.

### 🧾 `-wipe @пользователь confirm`
Безвозвратно стирает всю историю лицензий пользователя из базы данных.
Без `confirm` — только показывает, что будет удалено.
**Доступ:** только участники с правом **Администратор**.

### 🗑 `-del <количество>`
Удаляет последние N сообщений в текущем канале (лимит — 1000, кулдаун 5 сек).
**Доступ и каналы:** см. таблицу выше.

### 🧹 `-delus @пользователь <количество>`
Удаляет последние N сообщений пользователя по всем каналам сервера
(лимит — 500, кулдаун 10 сек). Проверяется канал, где **вызвана** команда.
**Доступ и каналы:** см. таблицу выше.

### 🤖 `-bot`
Информация о боте: разработчик, версия, дата обновления.
**Доступ:** всем.

### 📖 `-help`
Показывает список команд — каждому пользователю только те, к которым у него
есть доступ.
**Доступ:** всем (список команд разный в зависимости от ролей).

### 🛠 `-admin` — кнопочная админ-панель (🧪 BETA)
Открывает интерактивную панель управления: база данных, выдача/отзыв
лицензий, заметки, wipe истории, удаление сообщений — всё на кнопках,
без текстовых команд. Подробности — в разделе выше.
**Доступ:** роли `1520321736673853512`, `1520321736673853511`,
`1520321736673853509`, `1520321736648949932`.
**Управление:** только тот, кто вызвал команду.

## Изменённые файлы

- **`config.py`** — новые константы: `DELETE_GLOBAL_ROLE_IDS`,
  `DELETE_EXEMPT_ROLE_IDS`, `DELETE_NO_EXEMPT_ROLE_IDS`, `DELETE_CATEGORY_ID`,
  `DELETE_EXEMPT_CATEGORY_ID`, `LICENSE_COMMANDS_CHANNEL_ID`,
  `ADMIN_PANEL_ALLOWED_ROLE_IDS`, `ADMIN_PANEL_TIMEOUT_SECONDS`, `COLOR_ADMIN`.
- **`bot.py`** — обновлена `has_delete_access()` (теперь принимает канал и
  проверяет категории), добавлена `in_license_commands_channel()`,
  проверки канала добавлены в `-license`, `-cancel`, `-data`, обновлены
  тексты `-help`, добавлена команда `-admin`.
- **`admin_panel.py`** *(новый файл)* — вся логика кнопочной панели:
  главное меню, разделы, Modal-окна ввода данных, проверка прав и владельца
  панели.

`cooldown.py` и `license_db.py` не менялись.
