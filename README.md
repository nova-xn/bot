# Neway RP — Discord-бот модерации

Профессиональный бот для управления РП-проектом **Neway RP**. Работает исключительно на сервере с ID `1520321736611070013` — на любых других серверах команды полностью отключены.

## 📋 Возможности бота

### 🔹 Автовыдача ролей
При входе любого нового пользователя на сервер ему автоматически выдаётся роль `1521077456679731200`. Никаких действий не требуется — всё происходит автоматически по событию `on_member_join`.

### 🔹 Команда `-report @пользователь`
**Доступно:** всем участникам сервера  
**Кулдаун:** 60 сек (7 сек для привилегированных ролей)

Создаёт детальный репорт на пользователя:
- Собирает последние **20 сообщений** пользователя со всего сервера (с кликабельными ссылками)
- Извлекает информацию профиля:
  - Дата создания аккаунта
  - Дата вступления на сервер
  - Текущие роли
- Отправляет оформленный отчёт в канал логов `1521050858719154226`
- Автор получает подтверждение об успешной отправке

### 🔹 Команда `-cancel @пользователь`
**Доступно:** только модерации (роли ниже)  
**Кулдаун:** 60 сек (7 сек для привилегированных ролей)

**Разрешённые роли:**
- `1520321736673853512`
- `1520321736673853511`
- `1520321736673853510`
- `1520321736673853504`
- `1520321736673853503`
- `1520321736648949936`
- `1520321736648949935`
- `1520321736648949932`

**Что делает:**
- Снимает с пользователя роли `1520321736631914679` и `1520321736631914678` (если они есть)
- Отправляет лог выполнения в канал `1521052020633505912`

### 🔹 Команда `-del <количество>`
**Пример:** `-del 100`  
**Доступно:** модерации (роли ниже)  
**Кулдаун:** 5 сек (фиксированный)

**Разрешённые роли:**
- `1520321736673853512`
- `1520321736673853511`
- `1520321736673853510`
- `1520321736673853505`

**Что делает:**
- Удаляет указанное количество последних сообщений в текущем канале
- Максимум: **1000 сообщений** за вызов
- Сообщение с командой удаляется автоматически
- Ответ бота удаляется через 8 секунд
- Лог отправляется в канал `1521076264222588958`

### 🔹 Команда `-delus @пользователь <количество>`
**Пример:** `-delus @user 100`  
**Доступно:** тем же ролям, что и `-del`  
**Кулдаун:** 10 сек (фиксированный)

**Что делает:**
- Удаляет последние N сообщений указанного пользователя **по всем каналам сервера**
- Максимум: **500 сообщений** за вызов
- Автоматически обходит ограничения Discord на удаление сообщений старше 14 дней
- Подробный лог с разбивкой по каналам отправляется в `1521076264222588958`

⚠️ **Важно:** Discord API не позволяет массово удалять сообщения старше 14 дней через bulk delete. Для таких сообщений бот автоматически переключается на поштучное удаление, что медленнее, но работает.

## 🎯 Система кулдаунов

### Общий кулдаун (для `-report` и `-cancel`)
- **Обычные пользователи:** 60 секунд
- **Привилегированные роли** (`1520321736673853512`, `1520321736673853511`, `1520321736673853510`): 7 секунд

### Кулдауны команд удаления
- `-del`: **5 секунд** (фиксированный, отдельный от общего)
- `-delus`: **10 секунд** (фиксированный, отдельный от общего)
